# The Innovation Lab Creator

A field guide and working toolkit for designing, justifying and running a corporate innovation lab — built as the prep companion for the **emlyon Executive MBA × Exoflow Learning Expedition** (Lyon, 25 June 2026: Orange + Sanofi 14 Bis Innovation Lab).

**▶ Live site:** https://vbouin.github.io/innovation-lab-creator/

## What's inside
- **Immersive intro** — scroll-scrubbed video transitions and a video hero
- **Why labs · physical space · programs · roles · strategic fit · bottom-up intrapreneurship**
- **Innovation tools** toolbox (16 methods)
- **ROI calculator** — model a lab's economics across 6 project types
- **300 m² simulator** — drag-drop zoning & furniture → generates an exhaustive image-generation prompt from your 2D plan
- **Refined world map** — 106 innovation labs worldwide (success / mixed / cautionary)
- **100 labs table** — searchable, with the lesson from each
- **60-minute workshop** run-sheet, checklist, countdown, `.ics` export
- **A4 field-notes PDF** for note-taking during the visits ([download](Innovation-Lab-Field-Notes.pdf))

## Files
- `index.html` — the app (open directly or via the live site)
- `media/` — scrub videos + posters
- `Innovation-Lab-Field-Notes.pdf` — printable A4 note-taking booklet
- `build_pdf.py` — regenerates the PDF (`python3 build_pdf.py`, needs `reportlab`)

## Note
ROI figures and lab success rates are **illustrative teaching benchmarks**, not audited data.
